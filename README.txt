# Análisis y Predicción de Datos de Empleados

## Descripción
Este proyecto realiza el análisis y modelado de un dataset de empleados para predecir si un empleado tomará una licencia o no.

## Requisitos
- Python 3.x
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Instalación
Instala las dependencias necesarias usando pip:
```bash
pip install pandas matplotlib seaborn scikit-learn
